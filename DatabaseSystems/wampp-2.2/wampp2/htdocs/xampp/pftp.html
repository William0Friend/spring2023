FTP-Server (??)

Der Apache ist kein FTP-Server ...

... da er nun einmal das Protokoll HTTP (Hypertext Transfer Protocol) und eben nicht noch FTP (File Transfer Protocol) verwaltet.

In der wampp 2.2 Version ist aber ein wirklich freier AllroundServer namens "Fastream NetFile Server" beigelegt, der u.a. auch ein FTP Server beinhaltet. Zu finden im Verzeichnis [Laufwerk]:\wampp2\FastreamFTP. Doch bevor das Exekute "NetFileServer.exe" gestartet wird, bitte bei den folgenden Dateien den Laufwerksbuchstaben G: mit dem eurer aktuellen Partition (z.B. C:) ersetzen:

  • NetFileServer.ini
  • usersgroups.ini
  • user - Anonymous.fcf
  • user - xampp.fcf

Jetzt starten: Als Beispiel kann per FTP-Client sofort über die

Domain: 127.0.0.1
mit dem User: xampp
Passwort: wampp

auf das htdocs-Verzeichnis zugegriffen werden. Für Anonynmous User steht das Anonymous-Verzeichnis unter [Laufwerk]:\wampp2\anonymous bereit. Das ist nur eine Beispielkonfiguration, die es natürlich schon aus Sicherheitsgründen zu ändern gilt!

Der Faststream NetFile Server ist auch ein sehr einfacher HTTP Server, der in dieser Konfiguration auf Port 81 mit http://127.0.0.1:81 aufrufbar ist. Das Dokumentenroot für diesen kleinen Webserver befindet sich gleich Anonymous FTP unter [Laufwerk]:\wampp2\anonymous. Niemals den Fastream HTTP Server auf den Standardport 80 legen, da dann der Apache nicht mehr funktioniert. Merkregel: "Ein Server, ein Port"!
